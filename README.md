# MovieGuide
An app that lists popular/highest-rated movies. 

This app showcases MVP pattern and Uncle Bob Martin's Clean Architecture approach.

Libraries used:
 - RxJava
 - RxAndroid
 - OkHttp
 - Glide

Replace `API_KEY` in `Api.java` with your themoviedb.org API key.
